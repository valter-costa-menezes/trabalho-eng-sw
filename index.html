  O que é paradigma de programação ?

Paradigma de programação é um meio de se classificar as linguagens de programação baseado em suas funcionalidades Alguns dos principais paradigmas utilizados hoje no mercado:

  • Funcional
  • Lógico
  • Declarativo
  • Imperativo
  • Orientado a objetos
  • Orientado a eventos

  O que é GIT e GitHub ?

GIT

Git é a ferramenta que vai auxiliar o desenvolvedor a ter um histórico do seu processo de escrita de código.

Por meio desse software, é possivel visualizar as várias versões que o projeto já teve a partir de comandos.

GitHub

GitHub é uma plataforma que também permite o usúario ter o controle de todas as edições do código fonte.

O funcionamento do GitHub se destaca pelo armazenamento em nuvem, opção de copiar o código de outro repositório, fazer alterações e aceitar as mudanças sugeridas por outro desenvolvedor. Tudo isso simultaneamente.
